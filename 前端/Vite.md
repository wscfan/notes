# Vite

## 一、项目构建

### （一）构建Vite项目

1. NPM

   ```bash
   npm create vite@latest
   ```

   

2. Yarn

   ```
   yarn create vite
   ```

   

3. PNPM

   ```bash
   pnpm create vite
   ```

   

4. Bun

   ```bash
   bun create vite
   ```

   

### （二）构建Vite + Vue项目

1. NPM

   ```bash
   npm create vite@latest my-vue-app -- --template vue
   ```

   

2. Yarn

   ```bash
   yarn create vite my-vue-app --template vue
   ```

   

3. PNPM

   ```bash
   pnpm create vite my-vue-app --template vue
   ```

   

4. Bun

   ```bash
   bun create vite my-vue-app --template vue
   ```
