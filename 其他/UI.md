# UI笔记

## 一、UI设计基础

### 1. 分辨率
分辨率分为图像分辨率和显示/屏幕分辨率。
显示分辨率（dpi）:
&emsp;&emsp;1. 是指屏幕图像的精密度，也就是显示器所能显示的像素有多少。
&emsp;&emsp;2. 屏幕分辨率越高，所呈现的色彩越多，清晰度越高。
图像分辨率（ppi）：
&emsp;&emsp;1. 是指每英寸图像内的像素点数。

### 2. Web常用尺寸
UI设计分辨率： 72ppi
分辨率 = 屏幕X轴上的像素值 × Y轴上的像素值
目前普遍最小的电脑显示器分辨率： 1024px × 768px
内容长度无限制，宽度≤1024px，常用 1000px、 980px、 960px、 940px。
8pt ≤ 内容字号 ≤ 14pt。
